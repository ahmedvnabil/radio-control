تسجيل الدخول للمتصفح الحالي

قم بإدخال الـ Token هنا لفتح الصلاحيات لك في هذا المتصفح.

كيفية الاستخدام في التطبيقات الخارجية

قم بتمرير التوكن بإحدى طريقتين:

1. Via HTTP Header (Recommended)
curl -H "X-Guide-Token: sk_..." https://studio.naromx.org/api/v1/config
2. Via URL Query Parameter
curl "https://studio.naromx.org/api/v1/config?token=sk_..."
