Radio Control Engine

إدارة الحماية وتصاريح التطبيقات (Tokens)

لوحة تحكم Radio Control وواجهات الـ API محمية بالكامل لمنع الوصول غير المصرح به. يمكنك إدارة تصاريح الدخول للمتصفحات الحالية والأنظمة الخارجية من هنا.

تسجيل الدخول للمتصفح الحالي

أدخل توكن المسؤول (Guide Token) لتفعيل الصلاحيات الكاملة واستعراض البيانات الحية وسجلات النظام في هذا المتصفح.

كيفية الاستخدام في الأنظمة والإضافات الخارجية

عند إرسال طلبات من سكريبت أو تطبيق خارجي، قم بتمرير التوكن بإحدى طريقتين:

1. ترويسة الطلب HTTP Header (موصى به)
cURL Header
curl -H "X-Guide-Token: sk_..." https://studio.naromx.org/api/v1/config
2. معلمة عنوان الرابط Query Parameter
cURL Query
curl "https://studio.naromx.org/api/v1/config?token=sk_..."

مفاتيح الوصول الخارجية (API Tokens)

قم بتوليد مفاتيح وصول فريدة للإضافات والأنظمة الخارجية لتتحكم بصلاحية كل منها بشكل منفصل.

تم توليد التوكن بنجاح!

احفظ هذا التوكن في مكان آمن الآن، فلن تتمكن من رؤيته مجدداً لحماية أمان سيرفرك:

الاسم / الوصف البادئة (Prefix) تاريخ الإنشاء آخر استخدام الإجراءات
لا توجد أي توكنز نشطة حالياً. قم بإنشاء توكن للبدء.
