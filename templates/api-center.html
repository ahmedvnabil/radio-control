Radio Control Backend

إدارة الحماية والتصاريح (Tokens)

لوحة التحكم والـ API محميان بالكامل ولا يسمحان بأي طلبات خارجية إلا بوجود توكن صحيح.

تسجيل الدخول للمتصفح الحالي

قم بإدخال التوكن (GUIDE_TOKEN) هنا. سيتم حفظه كـ Cookie في متصفحك لكي تستطيع استخدام لوحة التحكم الرئيسية وتجربة الطلبات بدون الحاجة لكتابته كل مرة.

كيفية مناداة الـ API من الخارج (External Apps)

للتطبيقات الخارجية، يمكنك تمرير التوكن بإحدى طريقتين:

1. Via HTTP Header (Recommended)
curl -H "X-Guide-Token: YOUR_TOKEN" https://studio.naromx.org/api/v1/config
2. Via URL Query Parameter
curl "https://studio.naromx.org/api/v1/config?token=YOUR_TOKEN"
